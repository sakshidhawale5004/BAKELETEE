⚡ KEY REMINDERS

⚠️  CRITICAL:
    • Upload CONTENTS of dist/, not the dist folder
    • .htaccess must be in public_html/ root
    • Enable SSL certificate (HTTPS required)
    • Use https://, not http://

✅ BEST PRACTICES:
    • Verify file structure matches documentation
    • Test in multiple browsers
    • Check on mobile device
    • Monitor browser console (F12)
    • Backup important files

❌ DO NOT:
    • Upload node_modules/
    • Upload src/ folder
    • Expose .env files
    • Use HTTP without redirecting to HTTPS
