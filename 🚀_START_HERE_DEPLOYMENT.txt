🎯 QUICK START (3 SIMPLE STEPS)

STEP 1: Login to Hostinger
   → https://www.hostinger.com/
   → Enter your credentials

STEP 2: Upload Files
   → Open File Manager
   → Go to public_html/
   → Upload contents of ./dist/ folder
   → ⚠️  Upload contents of dist, NOT the dist folder itself

STEP 3: Enable SSL & Test
   → Enable SSL certificate in settings
   → Wait 5-10 minutes
   → Visit https://yourdomain.com
   → ✅ Done!

📦 WHAT YOU'RE UPLOADING

Location: ./dist/ folder
Size: ~20 MB

Contents:
✓ index.html (main page)
✓ assets/ (CSS, JavaScript, images)
✓ .htaccess (IMPORTANT - enables React Router)
✓ Images & favicons
✓ Public files

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⚠️  CRITICAL THINGS TO REMEMBER

1. Upload CONTENTS of dist/, not dist/ folder itself
   ❌ Wrong:  public_html/dist/index.html
   ✅ Right:  public_html/index.html

2. .htaccess file is ESSENTIAL
   → Must be in public_html/ root
   → Enables React Router to work
   → Shows hidden files if not visible

3. Enable SSL Certificate
   → Go to SSL section in Hostinger
   → Click "Activate SSL"
   → Wait 5-15 minutes

4. Use HTTPS, not HTTP
   → https://yourdomain.com ✅
   → http://yourdomain.com ❌

❌ COMMON ISSUES & QUICK FIXES

Problem: Blank white page
→ Fix: Check browser console (F12) for errors

Problem: CSS/Images not loading
→ Fix: Verify assets/ folder uploaded to public_html/

Problem: Page shows 404 on refresh
→ Fix: Ensure .htaccess is in public_html/ root

Problem: Website very slow
→ Fix: Contact Hostinger - check server resources

Problem: SSL certificate not activating
→ Fix: Wait 15 minutes, refresh, or contact Hostinger

More issues? → Read DEPLOYMENT_GUIDE.md section 🔧 TROUBLESHOOTING
